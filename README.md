# AI CV Intelligence (AICV)

AI-powered CV analysis and recommendation system that helps users improve their resumes using a conversational AI assistant and semantic document understanding.

---

## 🚀 Overview

AICV is a full-stack AI application that allows users to:

- Upload their CV (PDF format)
- Ask questions about improving their resume
- Receive AI-generated, personalized recommendations
- Leverage a backend powered by AI agents and vector search (RAG-ready architecture)

The goal is to simulate a **career intelligence assistant** that provides actionable feedback on CV structure, content, and optimization for job applications.

---

## 🧠 Features

- 📄 CV upload (PDF support)
- 💬 AI-powered Q&A about your resume
- 🧠 Backend AI agent for recommendation generation
- 🔎 Vector database integration (Chroma-based architecture)
- 🎨 Modern UI built with Streamlit + custom CSS
- ⚡ Fast API communication between frontend and backend
